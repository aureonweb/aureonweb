Panel del Aula
Organiza el aula en módulos; dentro de cada módulo agregas lecciones (YouTube, Drive, PDF o enlace). Marca un módulo como Alumnos para que quede protegido con el código. Recuerda pulsar Publicar cuando termines.
Administra los banners que aparecen al abrir la página principal. El banner de promociones se abre automáticamente al entrar; el de membresía se abre con el botón MEMBRESÍA. Ambos se muestran como carrusel. En el campo Texto escribe con normalidad: selecciona palabras y usa los botones para dar formato, sin código.

Contenido para alumnos (premium)

El código de alumnos es la clave que compartes con tus alumnos de paga. Con él se cifran los enlaces del contenido protegido: sin el código, nadie puede abrirlos, ni siquiera viendo el archivo del sitio.

Cambiaste el código. Al publicar, el contenido de alumnos se volverá a cifrar con el código nuevo y los alumnos deberán usar el nuevo código.

Configuración rápida

¿Te compartieron un código de configuración? Pégalo aquí y pulsa Activar. Con eso el botón Publicar queda listo — no necesitas llenar nada más.
